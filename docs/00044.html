Running The Application

Running the Application : pmsm_foc_pll_estimator_sam_e70_mchv3

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_encoder_sam_e70\firmware\mchv3_sam_e70_pim.X
  2. Clean and Build Project:

  1. Program the project:

  1. Press the switch "Push Button" to start/stop the motor.
  2. Vary Potentiometer to change the speed of the motor.

 

 

Running the Application : pmsm_foc_encoder_sam_e70 - MCLV2

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_encoder_sam_e70\firmware\mclv2_sam_e70_pim.X
  2. Clean and Build Project

  1. Program the project

  1. Press the switch S2 to start the motor.
  2. Vary Potentiometer to change the speed of the motor in clockwise direction.
  3. Press the switch S2 to stop the motor.
  4. Press the switch S3 to change the motor direction.
  5. Vary Potentiometer to change the speed of the motor in counter-clockwise direction.
