pmsm_foc_smo_sam_e70

The permanent magnet synchronous motors ( PMSM ) is widely used in various industries due to its high power density, smaller size and high efficiency. The Field oriented control is one of the most popular control mechanisms for the PMSM motor for applications which requires high dynamic performance. 

This example shows how to configure motor control peripherals of SAME70 for the control operation. The control strategy used is sensor-less FOC, in which rotor position and speed is determined by the Sliding Mode Observer ( SMO ). 

The following sections describe the basics of the SMO based FOC motor control algorithm and the steps to build, configure and run the PMSM motor for low voltage PMSM motor drive.
