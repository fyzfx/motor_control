Running The Application
  1. Build and Program the application using MPLAB X IDE.
  2. Connect to the RS232 port and configure the serial settings of X2CScope as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
  3. Observe the output as described in the following table:
Project Name 
Output 
mchv3_sam_e70_pim.X 

  1. Press the push button to start the motor.
  2. Vary Potentiometer to change the speed of the motor.
  3. Observe waveforms on X2CScope scope view.
  4. Press the push button to stop the motor.

 

mclv2_sam_e70_pim.X 

  1. Press the Switch S2 to start the motor.
  2. Vary Potentiometer to change the speed of the motor.
  3. Observe waveforms on X2CScope scope view.
  4. Press the Switch S2 to stop the motor.
