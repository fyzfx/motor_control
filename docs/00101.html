Hardware Setup
  1. Project mchv3_sam_e70_pim.X.
    • Hardware Used
      • MCHV3 Development Board (High Voltage)
      • Isolated Embedded Debugger Interface
      • ATSAME70 PIM
      • Leadshine EL5-M0400-1-24 Motor

 

  • Hardware Setup
    1. Mount the SAME70 144-pin Motor Control PIM on U9 header of the dsPICDEM MCHV-3 Development Board.

 

 

 

  1. Connect the three-phase wires of the PMSM motor to M1, M2 and M3.

 

 

 

  1. Connect External Op amp configuration Matrix board to header J4.

 

 

  1. Connect Isolated EDBG card as shown in the following figure. Refer to Isolated EDBG Card Information Sheet for instructions.

 

 

  1. Connect the jumpers J11, J12, J13 and J14.

 

 

  1. Connect the AC mains using J1.

 

 

  1. This application supports communication with the X2C-Scope plug-in to monitor and plot any variables used in the application. The communication between the target and the X2C-Scope plug-in can be established using the RS-232 serial port. Refer to the jumper positions marked on the development board chassis to select between the RS-232 serial port (J8) and the mini-USB port (J6). Refer to Using the X2C-Scope Plug-in for details on how to use the X2C-Scope in this demonstration.
