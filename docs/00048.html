Running The Application

Running the Application : pmsm_foc_encoder_position_sam_e54 - MCLV2

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_encoder_position_sam_e54\firmware\mclv2_sam_e54_pim.X
  2. Clean and Build Project

 

  1. Program the project

 

  1. Put the potentiometer knob at mid-point for zero position.
  2. Press the switch S2 to start/stop the motor.
  3. Rotate the potentiometer knob for clockwise position control, and anti-clockwise for anti-clockwise position control.

 

 

Fault Indication and Recovery : pmsm_foc_encoder_position_sam_e54 - MCLV2 

If LED D17 is ON, it indicates a non-recoverable fault caused by over-current condition. The MCU needs to be reset in order to recover from this fault mode. 

Project Name 
Output 
mclv2_sam_e54_pim.X 

  1. Press the Switch S2 to start the motor.
  2. Vary Potentiometer to change the reference rotor position of the motor.
  3. Observe waveforms on X2CScope scope view.
  4. Press the Switch S2 to stop the motor.

 

Note: The range of rotor position control is in terms of the number of encoder pulses.
