Running The Application

Running the Application : bldc_bc_hall_sam_c21

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\bldc_bc_hall_sam_c21\firmware\mclv2_sam_c21_pim.X.
  2. Clean and Build Project:

 

3. Program the project: 

 

4. Press the switch S2 to start/stop the motor. 

5. Press the switch S3 to toggle the motor direction. 

 

LED D2 Status 
Description 
OFF 
Motor spin direction is "forward" 
ON 
Motor spin direction is "reverse" 

NOTE: Direction toggle command is accepted only when the motor is stationary. 

  1. Vary the Potentiometer to change the speed of the motor.

 

Fault Indication and Recovery : 

If LED D17 is ON, it indicates a non-recoverable fault caused by over-current. The MCU needs to be reset in order to recover from this fault mode. 

Project Name 
Output 
mclv2_sam_c21_pim.X 

  1. Press the Switch S2 to start the motor.
  2. Vary Potentiometer to change the speed of the motor.
  3. Observe waveforms on X2CScope scope view.
  4. Press the Switch S2 to stop the motor.
