Hardware Setup
  1. Project mchv3_sam_e70_pim.X.
    • Hardware Used
      • MCHV3 Development Board (High Voltage)
      • Isolated Embedded Debugger Interface
      • ATSAME70 PIM
      • Leadshine EL5-M0400-1-24 Motor
    • Hardware Setup
      1. Mount the SAME70 144-pin Motor Control PIM on the dsPICDEM MCHV-3 Development Board, as shown in the following figure.
      2. Connect the three-phase wires of the PMSM motor to M1, M2 and M3 ports.
      3. Connect External Op amp configuration Matrix board to header J4.
      4. Connect Isolated EDBG card as shown in the following figure. Refer to Isolated EDBG Card Information Sheet for instructions.
      5. Connect the AC mains using J1.
      6. This application supports communication with the X2C-Scope plug-in to monitor and plot any variables used in the application. The communication between the target and the X2C-Scope plug-in can be established using the RS-232 serial port. Refer to the jumper positions marked on the development board chassis to select between the RS-232 serial port (J8) and the mini-USB port (J6). Refer to Using the X2C-Scope Plug-in for details on how to use the X2C-Scope in this demonstration.

 

 

 

  1. Project mclv2_sam_e70_pim.X.
    • Hardware Used
      • MCLV2 Development Board (Low Voltage)
      • ATSAME70 PIM
      • Long Hurst DMA0204024B101 Motor
    • Hardware Setup
      1. Mount the SAME70 144-pin Motor Control PIM on the dsPICDEM MCLV-2 Development Board, as shown in the following figure.
      2. Connect the External Op amp Configuration Matrix Board to header J14.
      3. Connect the three-phase wires coming from P1 header of the long Hurst motor (i.e., Red, White and Black to M1, M2 and M3 ports,respectively). Leave the green wire from the motor unconnected.
      4. Apply 24V DC at BP1-BP2/J2.
      5. This application supports communication with the X2C-Scope plug-in to monitor or plot any variables used in the application. The communication between the target and the X2C-Scope plug-in can be established using the RS-232 serial port (J10). To use the RS-232 serial port, connect JP4 and JP5 to the UART position.
      6. Refer to Using the X2C Scope for details on how to use X2C Scope in this demonstration.
