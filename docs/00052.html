Running the Application : pmsm_foc_pll_estimator_pic32_mk - MCLV2

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_pll_estimator_pic32_mk\firmware\mclv2_pic32_mk_pim.X
  2. Clean and Build Project

  1. Program the project

  1. Press the switch S2 to start/stop the motor.

5. Press the switch S3 to toggle motor direction.

LED D2 Status 
Description 
OFF 
Motor spin direction is "positive" 
ON 
Motor spin direction is "negative" 

NOTE: Direction toggle command is accepted only when the motor is stationary. 

  1. Vary the Potentiometer to change the speed of the motor.
