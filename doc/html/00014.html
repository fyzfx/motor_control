Running The Application
  1. Build and Program the application using MPLAB X IDE.
  2. Connect to the RS232 port and configure the serial settings of X2CScope as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
  3. Press the switch S2 to start the motor.
  4. Vary Potentiometer to change the speed of the motor.
  5. Press the switch S2 to stop the motor.
